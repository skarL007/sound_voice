Why VoiceLaunch

A voice that shows up where you actually talk.

Built for people who can't or prefer not to speak, for gamers who want a voice in chat, and for anyone who needs fast, private, assistive communication.

Virtual mic for Discord & games

The generated voice routes into a virtual microphone (VB-Cable, bundled in the installer). Any app that reads a mic hears it — Discord, Zoom, in-game voice chat.

Smart Auto routing

Online Edge TTS when you have internet, instant fallback to a local voice (Kokoro/Piper) when you're offline or the connection drops. No setting to flip.

Instant, no account

Hundreds of natural Microsoft Edge voices work the moment you open the app. Nothing to download, no sign-up, no telemetry.

Global voice shortcuts

Bind a phrase to a hotkey and fire it from inside a game without alt-tabbing. "GG", "cover me", or your own lines — one keypress.

Built for accessibility

Persistent history and draft, quick phrases, an on-screen keyboard, high-contrast and large-font modes, full keyboard navigation, visible focus.

Free & open source

MIT licensed, no paywall, no data collection. Local synthesis never leaves your machine. Contributions welcome.

How it works

Type → route → speak into any voice chat.

Auto mode picks the best engine per phrase and fans the audio to your virtual mic so Discord and games hear it — while you still hear yourself.

1 · You
Type a phrase or fire a shortcut
2 · Auto route
Edge online · Kokoro/Piper offline
3 · Virtual mic
CABLE Input (VB-Cable)
4 · They hear
Discord, Zoom & games
Voices

Cloud quality by default, local when you need it.

Auto mode selects the local engine by your hardware. The recommended local path is Piper first, Kokoro for higher quality.

EngineStatusNotes
Edge TTS (cloud)DefaultHundreds of voices, no install, needs internet
PiperStable · localLightweight, CPU-only, works offline
KokoroStable · localHigher quality (MOS 4.2), still CPU-friendly

Install

Running in three steps.

Windows 10/11 x64. The virtual microphone is bundled and sets itself up.

1

Download

Grab the latest VoiceLaunch-TTS-Setup.exe from GitHub Releases and run it. VB-Cable installs automatically.

2

Enable the mic

Open the app, turn on the virtual mic (it auto-selects CABLE Input), type a phrase, and hit Speak.

3

Point Discord at it

In Discord (or any game) set the input device to CABLE Output. Everyone in the channel now hears your voice.

Windows SmartScreen: this is an unsigned open-source build, so Windows shows "Windows protected your PC." Click More info → Run anyway. Every release publishes the installer's SHA-256 so you can verify it first: 
Get-FileHash -Algorithm SHA256 .\VoiceLaunch-TTS-Setup.exe
