# Archived documents

These are historical, mostly Portuguese-language working documents from the
MVP development phase — session checkpoints, audits, brainstorming plans, and
superseded design specs. They are kept for provenance but are **not** current
documentation.

For current docs, see the top-level [README](../../README.md),
[ARCHITECTURE](../ARCHITECTURE.md), and the files in [docs/](../).

Notably superseded:
- `superpowers/specs/2026-05-11-tech-noir-accessible-core-redesign-design.md`
  describes an old "tech noir" (cyan/Oxanium/glow) visual language that was
  **replaced** by the sober design v2 the app ships today. Do not use it as a
  design reference.
